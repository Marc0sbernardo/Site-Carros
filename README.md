# 🚗 FP Selection — Site de Carros

Site de exposição de carros de luxo com slider interativo e design moderno.

---

## 📋 Sobre o Projeto

O **FP Selection** é uma landing page desenvolvida para exibir carros de alto padrão — Porsche, Ferrari e Lamborghini — em um carrossel animado e visualmente impactante. O projeto foca em uma experiência de usuário fluida, com transições suaves e uma identidade visual elegante em tons escuros e roxo neon.

---

## ✨ Funcionalidades

- **Slider interativo** com navegação por botões (anterior / próximo)
- **Animações CSS** com transições encadeadas por delay nos elementos de conteúdo
- **Indicadores visuais** de slide ativo (pontos e numeração)
- **Layout fixo em tela cheia** com header fixo
- **Design responsivo** baseado em unidades viewport (`vw`, `vh`)

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica da página |
| CSS3 | Estilização, animações e layout com Grid |
| JavaScript (Vanilla) | Lógica do slider interativo |
| Google Fonts | Tipografia com a fonte **Poppins** |

## 🚀 Como Executar

Não é necessária nenhuma instalação. Basta seguir os passos:

1. Clone o repositório:
   ```bash
   git clone https://github.com/Marc0sbernardo/Site-Carros.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Site-Carros
   ```

3. Abra o arquivo `index.html` no seu navegador.

> **Dica:** Para uma melhor experiência, utilize a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) do VS Code.

---

## 🎨 Design

- **Paleta de cores:** Fundo preto, gradiente roxo/violeta, destaque verde neon (`#beff1b`)
- **Tipografia:** Poppins (Google Fonts)
- **Layout:** CSS Grid no header; posicionamento absoluto no slider
- **Imagens dos carros:** Exibidas com rotação em `-20deg` no estado ativo, criando um efeito dinâmico
